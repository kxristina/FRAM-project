FRAM - Local Farm Produce Delivery

FRAM is a responsive web application that connects customers with local farms and fresh seasonal produce. Users can browse products, learn about the delivery process, and locate participating farms through an interactive Google Maps integration.

Features:
Responsive design for desktop, tablet, and mobile devices
Seasonal produce showcase
Product catalog with pricing information
Farm-to-home delivery information
Newsletter subscription form
Interactive Google Maps integration
Modern and clean user interface
Pages

1. Home Page
The home page includes:
Hero banner
Seasonal product highlights
Information about how the service works
Popular produce section
Newsletter signup form

2. Products Page
The locations page displays:
Interactive Google Map
Farm locations

Installation:

Clone the repository
Navigate to the project folder
Open the project in your browser

Google Maps API Setup
The Locations page requires a Google Maps API key.
Step 1: Create a Google Cloud Project
Visit Google Cloud Console
Create a new project
Enable:
Maps JavaScript API
Places API (optional)
Step 2: Generate an API Key
Navigate to APIs & Services → Credentials
Create an API Key
Restrict the key to your domain for security
Step 3: Add the API Key by replacing "YOUR_API_KEY" in the html file
